❤️ Love Letter (Godot)
======================

A digital implementation of the classic **Love Letter** card game built with the Godot Engine.

This project explores turn-based game logic, UI interactions, and simple AI within a lightweight Godot architecture.

* * *

🎮 About the Game
-----------------

**Love Letter** is a fast-paced card game of deduction, risk, and luck.  
Players compete to deliver their letter to the princess while eliminating rivals.

Core mechanics:

*   Each player holds **one card**
*   Draw → play → resolve effect
*   Use card abilities to eliminate opponents or gain advantage
*   Last player standing (or highest card at round end) wins

* * *

✨ Features
----------

*   🃏 Core Love Letter gameplay mechanics
*   🤖 Basic AI opponents (bots)
*   🎯 Turn-based system with deterministic rules
*   🖥️ Clean 2D UI built in Godot
*   🔁 Round-based game loop
*   📦 Modular scene structure (cards, players, game controller)
